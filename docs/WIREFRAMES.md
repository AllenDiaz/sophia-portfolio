## Responsive Behavior

### Mobile (< 768px)
- Single column layout
- Hamburger navigation menu
- Stacked service cards
- Simplified hero with CTA at bottom
- Touch-friendly buttons (min 44x44px)
- Reduced text, focus on key info

### Tablet (768px - 1024px)
- Two-column layout for services
- Condensed navigation
- Adjusted spacing

### Desktop (> 1024px)
- Full multi-column layouts
- Expanded navigation
- Optimal line length for readability (60-80 characters)

---

## Design Specifications

### Spacing System
- **Section Padding**: 80px top/bottom (desktop), 48px (mobile)
- **Container Max-width**: 1280px
- **Content Max-width**: 800px for text-heavy pages
- **Grid Gap**: 24px between cards

### Color Usage
- **Primary Blue**: CTAs, links, icons
- **Teal**: Accents, hover states, secondary elements
- **Coral**: Important CTAs, attention-grabbing elements
- **Neutrals**: Text, backgrounds, borders

### Typography Scale
- **H1**: 48px (desktop), 32px (mobile), Geist Sans Bold
- **H2**: 36px (desktop), 24px (mobile), Geist Sans Semibold
- **H3**: 24px (desktop), 20px (mobile), Geist Sans Semibold
- **Body**: 16px, Inter Regular, line-height 1.7
- **Small**: 14px, Inter Regular

### Interactive Elements
- **Buttons**: 12px padding, rounded-lg (8px)
- **Cards**: Subtle shadow, hover elevation
- **Links**: Underline on hover, color transition
- **Forms**: Clear focus states, inline validation

## Conversion Optimization

### CTA Placement
1. **Hero Section**: Primary CTA above fold
2. **Service Section**: "View Services" or "Get Started"
3. **Mid-page CTAs**: After value propositions
4. **Footer CTA**: Final conversion opportunity

### Trust Signals
- HIPAA compliance badge
- Years of experience
- Certifications
- Client testimonials (future)
- Professional credentials

### Performance
- Hero images: < 100KB optimized
- Lazy load below-fold images
- Font preloading for FOIT prevention
- LCP target: < 2.5s
